A collection of D&D 5e tools built with React + TypeScript + Vite.

## What it does

- Browse and search D&D spells
- Filter by class, level, and school
- Mark spells as known
- Clean, responsive interface

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Lucide React

## Getting Started

```bash
npm install
npm run dev
```

## Planned Features

- Complete spell database
